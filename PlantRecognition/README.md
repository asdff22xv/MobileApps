# Распознавание растений (PlantRecognition)

Мобильное приложение для определения видов растений по фотографиям с использованием API PlantNet.org.

## Функционал
- 📸 Фотографирование растений на камеру
- 🖼 Загрузка изображений из галереи
- 🤖 Распознавание вида растения через PlantNet API
- 📋 Просмотр результатов с названием и вероятностью
- 📊 Информация о растении (научное и обычное название)

## Технологии
- **Язык:** Kotlin
- **Среда:** Android Studio
- **API:** PlantNet.org (https://my.plantnet.org/)
- **HTTP клиент:** Retrofit / OkHttp
- **Изображения:** Glide / Picasso
- **Минимальный API:** Android 7.0 (API 24)

## Архитектура приложения

Приложение построено на двух экранах. Главный экран позволяет сделать фото или выбрать изображение из галереи. После выбора фото отправляется на сервер PlantNet для распознавания. Результат отображается на втором экране с указанием вероятных видов растений и степенью уверенности.

### Компоненты системы

**Главный экран (MainActivity)** — содержит кнопки для запуска камеры и выбора фото из галереи, а также область предпросмотра выбранного изображения.

**Экран результатов (ResultActivity)** — отображает список распознанных видов растений с названиями и процентом совпадения.

**API клиент** — отправляет изображение на PlantNet API и получает JSON с результатами распознавания.

**Модель данных** — содержит структуру ответа от API: научное название, общее название, вероятность совпадения.

### Поток данных

Пользователь делает фото → изображение кодируется в Base64 → отправляется POST-запрос к PlantNet API → полученный JSON парсится → результаты отображаются на экране.

## Инструкция по запуску

1. Клонируйте репозиторий
2. Получите API ключ на https://my.plantnet.org/
3. Вставьте ключ в файл `MainActivity.kt`
4. Откройте проект в Android Studio
5. Синхронизируйте Gradle
6. Запустите на устройстве с камерой

## Инструкция по использованию

1. Нажмите "Сделать фото" или "Выбрать из галереи"
2. Дождитесь загрузки изображения
3. Нажмите "Распознать"
4. Просмотрите результаты — список возможных видов растений
