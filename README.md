# Мобильные приложения - Лабораторные работы

Лабораторные работы по дисциплине "Мобильная разработка"

## Проекты

### 1. Распознавание растений (PlantRecognition)
Приложение для распознавания видов растений по фотографиям с помощью камеры.



### 2. Контроль служебных автомобилей (CarControl)
Приложение для мониторинга использования служебного автотранспорта, учёта пробега и поездок.

[CAR](Screen_recording_20260514_195505-ezgif.com-video-speed.webm)


## Технологии
- Язык программирования: Kotlin
- Среда разработки: Android Studio
- Минимальная версия Android: 7.0 (API 24)
- Архитектура: MVVM
- Material Design
